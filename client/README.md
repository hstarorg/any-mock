# any-mock-client

``any-mock`` 的UI端项目，提供用户的注册登录以及APP、API的管理功能。

# How to develop?

本项目采用 ``vue`` 框架，结合 ``webpack`` 构建工具，主UI库采用 ``Bootstrap``。

要开始开发，请先fork项目，然后进入 client 目录，执行如下操作：

``` bash
# 安装依赖包
$ npm install

# 启动Server(支持热更新)
$ npm run start
# or
$ npm run dev

# 打包生产环境代码
$ npm run build
```
